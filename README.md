# Wetterprojekt

`git clone https://github.com/sean-imus/wetterprojekt`


## Learninglessons

Der Learninglessons Ordner enthält die Meilensteine die wir als Übung machen sollten.


## Setup

1. `1-downloader.py` um alle verfügbaren Dateien vom Deutschen Wetterdienst herunterzuladen
2. `2-extractor.py` um alle heruntergeladenen Dateien zu extrahieren
3. `3-create_db.py` um eine leere sqlite Datenbank mit dem korrekten Schema zu erstellen
4. `4-csv_to_sql.py` um alle CSV Dateien aus den extrahierten Ordnern in die Datenbank zu laden
