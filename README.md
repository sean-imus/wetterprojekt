## Setup

1. downloader.py um alle verfügbaren Dateien vom Deutschen Wetterdienst herunterzuladen
2. extractor.py um alle heruntergeladenen Dateien zu extrahieren
3. create_db.py um eine leere sqlite Datenbank zu erstellen mit dem korrekten Schema
4. csv_to_sql.py um alle CSV Dateien aus den extrahierten Dateien in die Datenbank zu laden 
5. main.py um mit der Datenbank zu interagieren
