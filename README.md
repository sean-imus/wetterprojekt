## Setup

1. Führe downloader.py aus um alle verfügbaren Dateien vom Deutschen Wetterdienst herunterzuladen
2. Führe extractor.py aus um alle heruntergeladenen Dateien zu extrahieren
3. Führe create_db.py aus um eine leere sqlite Datenbank zu erstellen mit dem korrekten Schema
4. Führe csv_to_sql.py aus um alle CSV Dateien aus den extrahierten Dateien in die Datenbank zu laden 
5. Führe main.py aus um mit der Datenbank zu interagieren
